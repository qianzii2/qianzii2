### 🚀 我最近在搞什么

#### [MBIS](https://crates.io/crates/mbis) – Memory‑Based Infinite Sampler
> 无限数据流 + 有限内存 → 好奇心驱动的有偏采样  
> **给异常数据更高的留存概率**，比随机采样聪明一丢丢

- ✅ RACE Sketch 密度估计  
- ✅ SIMD 加速（AVX2 + AVX-512）  
- ✅ Arena 内存管理 + 无锁队列  
- ✅ 自愈扩容 + Checkpoint 容错  
- ✅ 已发布到 crates.io：`cargo add mbis`

#### [RockDuck](https://github.com/qianzii2/rockduck) – HTAP 嵌入式数据库
> 一个**野心勃勃**的练习项目  
- MVCC + SSI 冲突检测  
- DeltaStore 行存 + Vortex 列存 + DuckDB SQL 引擎  
- 分布式实验分支（Gossip + 一致性哈希 + HLC）  

> ⚠️ 此项目主要用于“装逼”，但真心欢迎提 Issue
